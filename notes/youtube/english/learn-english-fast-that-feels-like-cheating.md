# Learn English FAST that feels like cheating

Daily Native English による英語学習を加速させるための8つのテクニック動画の要約。

## 8つの学習テクニック

### 7. 脳の栄養を整える
- 砂糖を避ける：砂糖はエネルギーラッシュの後に疲労が来て、集中力が低下する
- ブレインフード推奨：ナッツ、魚、ベリー、葉物野菜、アボカド
- 理由：言語学習は脳のパワーを消費するため、正しい燃料が必要

### 6. マインドセットの転換
- ネガティブ→ポジティブに変える例
  - 「50個単語を覚えないと」→「5個の実用的な単語をマスターしよう」
  - 「英語は難しい」→「ステップバイステップでマスターできる」
- 利点：ストレス軽減、脳が前向き指令に従う、学習効率向上

### 5. タイマーを使う（ポモドーロテクニック）
- 25分集中 → 5分休憩
- パーキンソンの法則：タスクに与えられた時間いっぱい使う
- タイマーのメリット：集中力向上、ゲーム感覚で楽しい、エネルギー維持

### 4. 答えから始める
- パターン学習：質問と答えのセットでリアルな会話パターンを習得
- 実用的：試験対策や会話に直接応用できる
- リソース：「400 Daily Questions and Answers」電子書籍の紹介

### 3. 重要なコンテンツを優先（80/20の法則）
- 20%の学習が80%の結果を生む
- 例：IELTS試験対策なら学術語彙に焦点
- 職場英語なら：会議表現、メール、プレゼン関連
- 結論：高頻出語彙に集中

### 2. 弱点に焦点を当てる
- 得意分野の繰り返しではなく、苦手な部分を直視する
- 例：発音や難しい文法ルール
- 原則：避けていることが最も成長できる領域
- 「ミスから学ぶ」ことが重要

### 1. AIツールを活用（ChatGPTなど）
- 24/7利用可能なパーソナルコーチ
- 会話練習、文法チェック、語彙改善
- リアルタイムフィードバック
- ジャッジなし、プレッシャーなし
- 利点：教師を待たずいつでも練習可能

## キーメッセージ
- 「勉強に費やす時間」より「どう勉強するか」が重要
- 楽しみながら学ぶこと
- 継続的な改善を目指す
