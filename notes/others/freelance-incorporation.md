# フリーランスの法人化まとめ

## 損益分岐点

**年間利益 800万〜1,000万円** が法人化を検討するライン。

| 利益 | 判断 |
|------|------|
| 500万円以下 | 個人事業主のまま有利 |
| 500〜800万円 | グレーゾーン（要試算） |
| 800万円超 | 法人化でメリットが出始める |

---

## 合同会社（LLC）vs 株式会社

| | 合同会社 | 株式会社 |
|--|----------|----------|
| 設立費用 | 約10万円 | 約25万円 |
| 決算公告 | 不要 | 必要（毎年） |
| 知名度 | やや低い | 高い |
| 上場 | 不可 | 可能 |

節税目的のみなら**合同会社で十分**。Apple Japan・Amazon Japan・Google Japan も合同会社。

---

## 有限責任とは

出資した金額以上の責任を負わない。

- 100万円出資 → 会社が1,000万円の借金を抱えても個人への請求は0円
- 個人事業主は無限責任 → 事業の借金 = 個人の借金

※ 銀行融資時に個人保証を求められると有限責任の恩恵は薄れる

## 法人化の主な節税メリット

### 1. 役員報酬 × 給与所得控除

自分に役員報酬を払うことで、法人の利益を減らしつつ個人側で給与所得控除が使える。

### 2. 税率の差

| 区分 | 実効税率 |
|------|----------|
| 個人（高収入時） | 約35〜50% |
| 法人（中小） | 約20〜34% |

### 3. 退職金（最強の節税）

- 退職金は会社の経費になる
- 勤続年数 × 40万円が非課税
- 残りの1/2だけ課税

例：20年後に5,000万円退職金 → 控除800万 → 残り4,200万の半分2,100万円だけ課税

---

## 税率比較シミュレーション（利益5,000万円）

### 個人事業主の場合

| 税目 | 金額 |
|------|------|
| 所得税（累進・最高45%） | 約1,770万円 |
| 住民税（10%） | 約500万円 |
| 事業税（5%） | 約235万円 |
| **合計** | **約2,500万円（実効約50%）** |

### 法人化 + 役員報酬800万円の場合

| 区分 | 税目 | 金額 |
|------|------|------|
| 会社（残り4,200万） | 法人税等（約30%） | 約1,300万円 |
| 個人（役員報酬800万） | 所得税+住民税+社保 | 約200万円 |
| **合計** | | **約1,500万円（実効約30%）** |

→ **約1,000万円の節税**

## 最適役員報酬

**年600〜800万円（目安700万円）** が多くの税理士の推奨値。

### 考え方

法人税率（800万円超）≈ 34%  
個人の実効税率がこれを超えない範囲で役員報酬を設定する。

| 役員報酬 | 判断 |
|----------|------|
| 600万円以下 | もっと増やした方が節税 |
| 600〜800万円 | 最適ゾーン |
| 900万円超 | 個人税率が高くなりすぎ |

### 法人税の境界線（800万円）

法人利益800万円以下：実効約21%  
法人利益800万円超：実効約34%

→ 役員報酬を「総利益 - 800万円」に近い水準に設定するのが基本戦略。

### 変動要因

- 扶養家族がいる → 控除が増えるので少し多めでもOK
- 社会保険最小化 → 月8.8万円（年約106万）に抑えて法人留保を最大化（老後年金は減る）
- 配偶者に役員報酬 → 所得分散でさらに節税

---

## 法人化のコスト

| 項目 | 金額 |
|------|------|
| 設立費用（合同会社） | 約10万円（初回のみ） |
| 税理士費用 | 年30〜50万円 |
| 法人住民税均等割 | 年約7万円（赤字でも発生） |
| 社会保険料 | 役員報酬設定で必須加入 |

---

## 注意点

- 役員報酬は**期首に1年間固定**（定期同額給与）。急な利益増でも変更不可
- 法人税は内部留保にも**まずかかる**（払った後に留保）
- 1人会社でも**社会保険加入が義務**
- 顧問税理士費用（年50万）でも数百万節税できれば余裕で元が取れる
