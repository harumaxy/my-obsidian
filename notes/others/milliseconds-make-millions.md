# ミリ秒の差が数百万ドル

出典: [web.dev - Milliseconds Make Millions](https://web.dev/case-studies/milliseconds-make-millions)

## 概要

Google の委託により 55 と Deloitte が実施した調査。欧米の大手ブランド 37 サイト・3,000 万件以上のセッションデータを分析。

**4 つの速度指標を 0.1 秒改善するだけで**、購入ファネル全体の進行率が向上する。

対象指標:
- First Meaningful Paint（現在は非推奨 → LCP 推奨）
- 推定入力レイテンシ
- 平均ページ読み込み時間（GA）
- 最大サーバーレイテンシ（TTFB）

## 業種別の効果

| 業種 | 主な改善効果 |
|------|-------------|
| **小売** | 商品詳細→カート追加 **+9.1%**、消費者支出 **+9.2%** |
| **高級品** | お問い合わせページへの進行率 **+20.6%**、商品詳細→カート追加 **+40.1%** |
| **旅行** | 購入完了率 **+2.2%**、予約率 **+10%** |
| **リード獲得** | フォーム送信ページへの進行率 **+21.6%**、ページビュー **+7%** |

## ポイント

- わずか **0.1 秒**の改善でも収益に大きなインパクトがある
- 高級品カテゴリは絶対値が低い分、改善効果の率が最も大きい
- モバイル体験の質がブランド間の競争力格差を拡大させていく
